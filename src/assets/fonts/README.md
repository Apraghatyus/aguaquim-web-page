# Fuentes de marca AGUAQUIM

Colocar aquí los archivos de fuente del logo (preferiblemente .woff2):

1. `IwataUDGothicDisplayPro-Heavy.woff2` → para "AGUA"
2. `UltineCond-Medium.woff2` → para "QUIM"
3. `EFHerbert-Regular.woff2` → para "LABORATORIO"

## Cómo obtenerlas

- Pedirle al diseñador del logo los archivos originales
- O comprarlas en https://www.myfonts.com/

## Formatos aceptados

Idealmente `.woff2` (máxima compresión). También se acepta `.woff`, `.otf` o `.ttf`.
Si tienes .otf/.ttf, conviértelos a .woff2 en: https://cloudconvert.com/ttf-to-woff2
